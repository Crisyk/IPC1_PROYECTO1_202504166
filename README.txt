README

BiblioSystem
Sistema de gestión de biblioteca desarrollado en Java con interfaz gráfica Swing.
Funcionalidades
Registro de usuarios
Gestión de libros
Control de préstamos
Historial de usuario
Requisitos
Java JDK 8 o superior
Ejecución
Compilar:
javac Main.java
Ejecutar:
java Main
Archivos necesarios
cuentas.txt
prestamos.txt
bitacora.txt (se genera automáticamente)
